# 使用指南

在真实场景中配置和操作 xbot 的实用指南。

## 可用指南

{{< columns >}}

### [沙箱](/zh-cn/guides/sandbox/)
配置 Docker 或远程沙箱隔离来执行 Shell 命令。了解沙箱模式、Docker 设置、远程 Runner 配置和安全注意事项。

### [权限控制](/zh-cn/guides/permission-control/)
设置基于 OS 用户的权限控制来管理工具执行。限制 Agent 可以以哪些用户身份执行命令，并支持特权操作审批流程。
